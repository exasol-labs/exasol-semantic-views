# SQL Examples

Example files:

- `sales_physical_model.sql`
- `sales_model_seed.sql`
- `sales_semantic_queries.sql`
- `sales_osi.yaml`

Run `sales_physical_model.sql` before `sales_model_seed.sql`.

`sales_osi.yaml` is an OSI `0.2.0.dev0` interoperability export for the sales
model, generated by:

```sh
python3 tools/osi.py export --model sales --object SALES --profile interoperability --output sql/examples/sales_osi.yaml
```

It is an import/export example, not an install script.
